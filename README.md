# Drone_RF_Detection_FPGA (RF Module)

## Overview
This repository contains the RF signal detection module of a larger drone project. It implements a real-time FPGA-based pipeline to detect drone RF signals in a low-cost, educational/experimental setting. The focus is on learning FPGA signal processing and RF detection algorithms, rather than long-range or military-grade detection.

## Tech Stack
- **Languages:** Python 3.12 (Modeling), VHDL-2008 (RTL)  
- **Tools:** Xilinx Vivado, GHDL, GTKWave, Cocotb  
- **Hardware Target:** AMD Kria K26 SOM  
- **RF Receiver:** HackRF One (2.4/5.8 GHz signals)
